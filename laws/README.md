# Laws

This folder documents the Greek (and EU) legal provisions that underpin the tax logic in this calculator.

| File | Law | What it covers |
|------|-----|----------------|
| [5246-2025_income-tax-brackets.md](5246-2025_income-tax-brackets.md) | Law 5246/2025 | Progressive income tax brackets effective 2026 |
| [5078-2023_ul-pension-exit-tax.md](5078-2023_ul-pension-exit-tax.md) | N.5078/2023 | Exit tax on unit-linked insurance / group pension payouts (from 2024) |
| [EFKA-2025_social-security-rates.md](EFKA-2025_social-security-rates.md) | EFKA 2025 | Employee (13.37%) and employer (21.79%) SS rates and annual cap |
| [4099-2012_ucits-capital-gains-exemption.md](4099-2012_ucits-capital-gains-exemption.md) | ν.4099/2012 + UCITS Directive 2009/65/EC | 0% capital gains tax on UCITS ETFs (e.g. VUAA) for Greek residents |

Each file describes the relevant provisions, how they map to code in `src/js/tax.js`, and links to official sources.
